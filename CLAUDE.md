# CLAUDE.md

## Project Overview

Personal portfolio website for Marcel Matschke, hosted at [matschke.dev](https://matschke.dev) via GitHub Pages.

## Stack

- Static HTML/CSS site (single `index.html`)
- Bootstrap 5 (local copy in `assets/bootstrap/`)
- Custom CSS in `assets/css/`
- No build tools or package manager

## Structure

```
index.html          # Main (and only) page
CNAME               # matschke.dev
assets/
  bootstrap/        # Bootstrap CSS/JS
  css/              # Custom styles (styles.min.css, fonts.css)
  fonts/            # Local font files
  img/
    avatars/        # Profile photo
    techstack/      # Tech stack SVG icons
    favivons/       # Favicons
  js/               # JavaScript
```

## Deployment

Push to `main` branch → auto-deploys via GitHub Pages.

## Language

Site content is in German.
